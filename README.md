# Craft management system #
# About
-  This is the system which is useful for Crafters to
 Manage the materials used for crafts.

# Features
-	Add craft materials.
-	Display all materials.
-	Search the material by Id.
-	Update Quantity. 
-	Delete craft materials.
-	Save the data using file handling.

# Execution 
-	Open any c compiler(Turbo c++,Code blocks,vs code etc.)
-	Run and compile the code.
-	Choose any feature from the options.
-	Follow the instructions displayed on screen.
-	Your data will be saved automatically.

# Language used
-	Programming language:C.

# Concept used
-	Array.
-	Structure.
-	Function.
-	File handling.
